Application Web de So-Pekocko

Piquante est une application de So-Pekocko qui vous permet de visualiser, créer et évaluer les sauces téléchargées par d'autres utilisateurs . 
Ce référentiel github représente la création d'une API privée dont le but sera de manipuler les informations des sauces ou de les stocker en fonction de l'authentification de l'utilisateur.


**********Installation**********

Prérequis : Vous devez avoir Node.js et node-sass d'installé sur votre machine.


1) Cloner ce repo sur votre machine.
2) Utiliser la commande "npm install" pour recuperer tout les modules nécessaires à l'application
3) A partir du dossier backend, lancer la commande "node server" pour demarrer la BDD.
4) A partir du dossier frontend, lancer la commande "ng serve" pour lancer l'application. Url de l'App ://localhost:4200/
